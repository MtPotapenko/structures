Вступ до HTML

HTML (HyperText Markup Language) - це мова розмітки, яка використовується для створення структури веб-сторінок. Вона складається з набору елементів, які оточують різні частини контенту, щоб відобразити його певним чином або діяти певним чином.

Основні поняття HTML включають:

  • Теги - основні будівельні блоки HTML
  • Атрибути - додаткова інформація про елементи
  • Вкладеність елементів
  • Пусті елементи (наприклад, <img>)

Структура HTML-документа

Типовий HTML-документ має чітку структуру:

<!DOCTYPE html>
<html>
<head>
    <title>Заголовок сторінки</title>
</head>
<body>
    <h1>Мій заголовок</h1>
    <p>Мій абзац.</p>
</body>
</html>

Кожен HTML-документ повинен починатися з декларації DOCTYPE, яка вказує браузеру, яку версію HTML використовує документ. Після цього йде кореневий елемент <html>, який містить два основних розділи: <head> (метаінформація) та <body> (вміст сторінки).

Семантичні елементи HTML5

HTML5 ввів нові семантичні елементи, які покращують структуру сторінки та її доступність:

  • <header> - вступний вміст або набір навігаційних посилань
  • <nav> - розділ з навігаційними посиланнями
  • <main> - основний вміст документа
  • <section> - тематичне угруповання контенту
  • <article> - незалежний, самодостатній вміст
  • <aside> - вміст, що має непряме відношення до основного
  • <footer> - нижній колонтитул для свого найближчого предка

Використання цих елементів покращує SEO та доступність вашого сайту, оскільки допомагає пошуковим системам та допоміжним технологіям краще зрозуміти структуру сторінки.

Вступ до CSS

CSS (Cascading Style Sheets) - це мова стилів, яка використовується для опису зовнішнього вигляду документів, написаних HTML. CSS дозволяє розділити вміст (HTML) та його представлення (стилі).

Основні переваги CSS:

  • Контроль зовнішнього вигляду багатьох сторінок з одного файлу
  • Більш складні макети, ніж з чистим HTML
  • Різні стилі для різних пристроїв (наприклад, друк або мобільні пристрої)
  • Покращена доступність

Селектори CSS

Селектори визначають, до яких елементів HTML будуть застосовані стилі. Основні типи селекторів:

Тип селектора Приклад Опис
Селектор елемента p { color: blue; } Застосовується до всіх елементів <p>
Селектор класу .highlight { background: yellow; } Застосовується до всіх елементів з class="highlight"
Селектор ID #header { font-size: 2em; } Застосовується до елемента з id="header"
Універсальний селектор * { margin: 0; padding: 0; } Застосовується до всіх елементів

Техніки макетування CSS

Сучасний веб-дизайн використовує кілька основних технік для створення макетів:

Flexbox

Flexbox - це одновимірна модель макету, яка дозволяє ефективно розподіляти простір між елементами в контейнері. Вона особливо корисна для компонентів інтерфейсу та дрібних макетів.

CSS Grid

CSS Grid - це двовимірна система макету, яка дозволяє створювати складні макети з рядками та стовпцями. Вона ідеально підходить для макетів сторінок.

Float

Техніка float раніше була основною для створення макетів, але зараз вона в основному використовується для обтікання текстом зображень.

Positioning

Властивість position дозволяє точно позиціонувати елементи відносно їх батьків або вікна браузера.
